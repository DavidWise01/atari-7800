# ATARI 7800

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **EXEREÚNESIS**.

ATARI 7800 (7800) — a UD0 EXEREÚNESIS teardown + spec edition. Atari's belated 2600 successor wrapped a 6502 around a smart graphics chip called MARIA that could throw far more sprites at the screen with little CPU help — and it kept the old TIA chip inside for full Atari 2600 backward compatibility. Good hardware, undone by arriving years late into Nintendo's era. Render-not-invent, two-layer (datasheet fact vs lore); no ROMs/BIOS/code; manufacturer cited not minted. LIVE clickable block-diagram teardown + spec sheet. Vellum (no black) + quorum-ring favicon. 6 components.
